# URL2Webhook

This is Chrome Extension, Throw URL into Webhook.

## Description

This Chrome Extension sends the URL of the currently open page to Webhook.

## Install

1. git clone  
```
git clone https://github.com/sk39kii/URL2Webhook.git
```

2. Load in Chrome

Click the Send button
